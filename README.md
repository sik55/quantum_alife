# Quantum ALife（量子人工生命）

## 概要

量子コンピューター（量子回路）を使用した二次元セルラー・オートマトン（量子ライフゲーム）です。

## 動作環境

Python 3.9  
NumPy 1.22.4  
Pandas 1.4.2  
Vispy 0.10.0  
PyQt 5.15.6  
Qiskit 0.36.2  
SciPy 1.8.1  
Matplotlib 3.5.2
